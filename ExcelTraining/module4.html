Module 4: The "Big 5" Aggregation Functions

SUM, AVERAGE, COUNT, MAX, MIN - Summarizing data instantly

üéØ Key Concepts

1. SUM - Adding It All Up

  • Syntax: =SUM(range)
  • Example: =SUM(A1:A10)
  • Can sum multiple ranges: =SUM(A1:A10, C1:C10)
  • Ignores text and blank cells

2. AVERAGE - Finding the Middle

  • Syntax: =AVERAGE(range)
  • Calculates arithmetic mean
  • Only includes cells with numbers
  • Blank cells are ignored, zeros are included

3. COUNT vs COUNTA - The Counting Functions

  • COUNT: Counts cells containing numbers only
  • COUNTA: Counts all non-empty cells
  • COUNTBLANK: Counts empty cells
  • Essential for data validation
  • COUNT(A1:A10) vs COUNTA(A1:A10)

4. MAX & MIN - Finding Extremes

  • MAX: Returns the largest value
  • MIN: Returns the smallest value
  • Ignore text and blank cells
  • Useful for finding best/worst performers
  • Example: =MAX(A1:A10)

5. Pro Tips for Aggregation Functions

  • Functions automatically expand ranges when you add data
  • Use Ctrl+Shift+End to select to the last cell
  • Functions ignore hidden rows (useful with filters)
  • Combine with IF for conditional aggregation

üí° Pro Tips

  • Use Alt + = for quick SUM function
  • Double-click function names in formulas to see help
  • Functions update automatically when data changes
  • Use absolute references ($A$1) when copying formulas
  • COUNTA is perfect for counting survey responses

üîç Common Mistakes to Avoid

  • Using COUNT when you need COUNTA (or vice versa)
  • Forgetting that AVERAGE ignores blank cells but includes zeros
  • Not understanding that MAX/MIN ignore text values
  • Selecting too small a range and missing data
  • Mixing data types in the same column

üìà When to Use Each Function

  • SUM: Total sales, expenses, quantities
  • AVERAGE: Mean performance, temperature, ratings
  • COUNT: Number of test scores, sales transactions
  • COUNTA: Survey responses, attendance records
  • MAX/MIN: Best/worst performance, price ranges
