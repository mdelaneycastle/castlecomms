üéØ Key Concepts

1. The Golden Rule of Formulas

EVERY formula in Excel must start with an equals sign (=)

  • Without =, Excel treats it as text
  • = tells Excel to calculate something
  • You can start with + or - but Excel adds = automatically

2. Basic Mathematical Operators

  • + (Addition): =A1+B1
  • - (Subtraction): =A1-B1
  • * (Multiplication): =A1*B1
  • / (Division): =A1/B1
  • ^ (Exponent): =A1^2 (A1 squared)
  • () (Parentheses): Control order of operations

3. The AutoSum Magic

  • Keyboard shortcut: Alt + =
  • Automatically detects range to sum
  • Works with rows and columns
  • Can be edited after insertion

4. Cell References in Formulas

  • Relative References: A1, B2 (change when copied)
  • Absolute References: $A$1 (never change)
  • Mixed References: $A1 or A$1
  • Press F4 to cycle through reference types

5. Order of Operations (PEMDAS)

Excel follows standard mathematical order:

  1. Parentheses
  2. Exponents
  3. Multiplication and Division (left to right)
  4. Addition and Subtraction (left to right)

üõí Practical Exercise: "Grocery Receipt Calculator"

Create a functional grocery receipt with automatic calculations:

  1. Create headers: Item, Price, Quantity, Total
  2. List 5 grocery items (Milk, Bread, Eggs, Apple, Cheese)
  3. Enter prices for each item
  4. Enter quantities purchased
  5. In Total column, create formula: =Price*Quantity
  6. Copy the formula down for all items
  7. Use AutoSum to calculate Subtotal
  8. Add Tax row: =Subtotal*0.10 (10% tax)
  9. Calculate Grand Total: =Subtotal+Tax
  10. Format all prices as currency

Bonus: Add a discount row that calculates 5% off the subtotal before tax

üí° Pro Tips

  • Use parentheses to make complex formulas clearer
  • Click cells instead of typing addresses when building formulas
  • Use F9 to preview formula results before confirming
  • Color-code cells: Excel shows matching colors when editing formulas
  • Name important cells (like tax rate) for easier reference

üîç Common Mistakes to Avoid

  • Forgetting the = sign at the beginning of formulas
  • Using x instead of * for multiplication
  • Not using parentheses for complex calculations
  • Circular references (a formula referring to itself)
  • Dividing by zero (results in #DIV/0! error)
