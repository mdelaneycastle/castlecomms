üéØ Key Concepts

1. Data Types in Excel

Excel recognizes different types of data automatically:

  • Text: Any combination of letters, numbers, and symbols (left-aligned by default)
  • Numbers: Numeric values for calculations (right-aligned by default)
  • Dates: Recognized date formats (stored as numbers internally)
  • Times: Hour and minute values (stored as decimal fractions)
  • Formulas: Start with = sign and perform calculations
  • Boolean: TRUE or FALSE values

2. The Magic Fill Handle

The Fill Handle is the small square in the bottom-right corner of a selected cell:

  • Auto-fill Series: Drag to continue patterns (1, 2, 3... or Mon, Tue, Wed...)
  • Copy Values: Ctrl+drag to copy the same value
  • Custom Lists: Excel recognizes days, months, and custom patterns
  • Double-click: Auto-fills down to match adjacent column length

3. Basic Formatting Tools

  • Font Formatting: Bold (Ctrl+B), Italic (Ctrl+I), Underline (Ctrl+U)
  • Alignment: Left, Center, Right, and Vertical alignment
  • Number Formats: Currency ($), Percentage (%), Comma (,), Decimal places
  • Borders: Add lines around cells for visual separation
  • Fill Color: Background colors for cells
  • Font Color: Change text color for emphasis

4. Adjusting Column Width & Row Height

  • Manual Adjustment: Drag the border between column/row headers
  • Auto-Fit: Double-click the border to fit content automatically
  • Precise Size: Right-click header ‚Üí Column Width or Row Height
  • Multiple Selection: Select multiple columns/rows to adjust together

5. Data Entry Best Practices

  • Use consistent formatting for similar data types
  • Enter dates in recognized formats (MM/DD/YYYY or DD-MMM-YYYY)
  • Start text with apostrophe (') to force text format for numbers
  • Use Tab to move right, Enter to move down while entering data
  • Press F2 to edit cell contents without clearing

üìÖ Practical Exercise: "Weekly Schedule"

Create a formatted weekly schedule to practice data entry and formatting:

  1. Create Headers:
    • In cell A1, type "Time"
    • In B1, type "Monday"
    • Select B1 and use the Fill Handle to drag to F1 (creating Mon-Fri)
  2. Add Time Slots:
    • In A2, type "9:00 AM"
    • In A3, type "10:00 AM"
    • Select both cells and drag down to create hourly slots until 5:00 PM
  3. Format the Schedule:
    • Select row 1 and make it Bold (Ctrl+B)
    • Center align all headers
    • Add borders around all cells
    • Make the header row a different color (fill color)
  4. Add Activities:
    • Fill in some activities (e.g., "Meeting", "Lunch", "Project Work")
    • Color-code different types of activities
    • Use different colors for Work (blue), Break (green), Meeting (yellow)
  5. Final Touches:
    • Auto-fit all columns to content
    • Increase row height for better readability
    • Add a title above the schedule in merged cells

üí° Pro Tips

  • Hold Ctrl while clicking to select non-adjacent cells
  • Press Ctrl+; to insert today's date
  • Press Ctrl+Shift+; to insert current time
  • Use Ctrl+D to fill down from the cell above
  • Use Ctrl+R to fill right from the cell to the left
  • Press Alt+Enter to add a line break within a cell
  • Format Painter (paintbrush icon) copies all formatting from one cell to others

üîç Common Mistakes to Avoid

  • Mixing data types in the same column (breaks sorting and formulas)
  • Using spaces for alignment (use proper alignment tools instead)
  • Forgetting that dates are actually numbers (important for calculations)
  • Over-formatting (too many colors and fonts make data hard to read)
  • Not using number formats (typing $ manually instead of currency format)

üìä Number Format Quick Reference

  • Ctrl+Shift+1 - Number format with commas
  • Ctrl+Shift+4 - Currency format
  • Ctrl+Shift+5 - Percentage format
  • Ctrl+Shift+6 - Scientific notation
